# Advanced_Express_API

### Express Generator + EJS 로 서버 사이드 렌더링을 구현했다.
https://www.themoviedb.org/ 의 API 를 이용해서 
1. 현재 상영중인 영화 목록
2. 개별 영화의 상세 정보
3. 검색 기능
을 직접 구현했다.
